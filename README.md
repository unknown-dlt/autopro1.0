# AutoPro — сервис для автосервиса

Веб‑приложение для управления записью клиентов, складом запчастей, механиками и отчётами.

## Структура проекта

- `frontend` — SPA на Vue 3 + Vite
- `backend` — простой API на Express, хранит данные в `backend/data.json`
- корень (`autopro`) — обёртка, позволяющая запускать фронтенд и бэкенд одной командой

## Требования

- **Node.js** v18+ (желательно LTS)
- **npm** (идёт вместе с Node)

## Первый запуск

1. Откройте терминал в корне проекта:

   ```bash
   cd c:\Users\bitho\autopro
   ```

2. Установите зависимости для корневого скрипта, фронтенда и бэкенда:

   ```bash
   cd backend && npm install
   cd ../frontend && npm install
   cd .. && npm install
   ```

3. Запустите приложение одной командой:

   ```bash
   npm run dev
   ```

   Эта команда:

   - поднимает **бэкенд** на `http://localhost:3000`
   - поднимает **фронтенд** (Vite) на `http://localhost:5173`

4. Откройте в браузере:

   ```text
   http://localhost:5173
   ```

   Данные сохраняются в файле `backend/data.json`.

## Отдельный запуск фронтенда и бэкенда

### Бэкенд

```bash
cd backend
npm install          # при первом запуске
npm run dev          # или npm start для обычного запуска
```

API будет доступен по адресу `http://localhost:3000`.

### Фронтенд

```bash
cd frontend
npm install          # при первом запуске
npm run dev
```

Vite откроет интерфейс по адресу `http://localhost:5173` (или подскажет другой порт в терминале).
